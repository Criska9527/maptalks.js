# maptalks.vt

maptalks's Vector Tile code base

## Install

Install lerna at first if you haven't.
```shell
npm i lerna -g
```

### Install maptalksgl

clone and install [maptalksgl](https://github.com/fuzhenn/maptalksgl) at the same folder of maptalks.vt

### Init
```shell
lerna bootstrap --hoist
```
